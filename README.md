# Pemrograman Mobile Flutter
Repository ini digunakan sebagai tempat pengumpulan seluruh hasil **Praktikum** dan **Tugas** untuk mata kuliah *Pemrograman Mobile* (Flutter).  
Karena setiap modul memiliki project yang berbeda, kami memisahkannya ke dalam **branch** agar lebih rapi dan mudah dicek.
